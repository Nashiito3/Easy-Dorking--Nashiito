# Installation

__Linux__

Clone the repo:
```
https://github.com/Nashiito3/Easy-Dorking--Nashiito.git
```
Go to the directory:
```
cd Easy-Dorking--Nashiito
```
Setup:
```
sudo bash install.sh
```
Install mdodules:
```
pip install -r requeriments.txt
```

__Windows__

Download as ZIP and extract.

Open terminal and type:
```
pip install -r requeriments.txt
```

Not tested in Termux


# Run

__Windows & Linux__

Type in terminal:
```
python3 main.py
```
